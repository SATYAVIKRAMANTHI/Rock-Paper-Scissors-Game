# Rock Paper Scissors Game

## Project Overview
This project is a command-line based Rock Paper Scissors game developed in C++ using Object-Oriented Programming (OOP) concepts. It allows users to play against the computer with an interactive menu-driven interface.

## Features
- Interactive command-line interface
- Rock, Paper, and Scissors gameplay
- Random computer moves
- Modular design using classes
- Object-Oriented Programming concepts

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Makefile
- VS Code

## Project Structure
- play_game.cpp - Main program
- RPSGame.cpp / RPSGame.hpp - Game logic
- Rock.cpp / Rock.hpp - Rock class
- Paper.cpp / Paper.hpp - Paper class
- Scissor.cpp / Scissor.hpp - Scissor class
- Tool.cpp / Tool.hpp - Base class
- makefile - Build configuration

## How to Run

Compile the project:

```bash
make
```

Run the executable:

```bash
./playgame
```
